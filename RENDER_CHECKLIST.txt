## üéØ What Each Setting Does

**`requirements-demo.txt`:**
- Only 3 packages (Flask, Gunicorn, python-dotenv)
- 50 MB install vs 1.5 GB
- **Much faster deployment**

**`gunicorn -b 0.0.0.0:$PORT portfolio_app:app`:**
- `portfolio_app` = your file name
- `app` = Flask object in that file
- `$PORT` = Render assigns this automatically

**`PORTFOLIO_DEMO_MODE=True`:**
- Uses pre-computed sample data
- No API calls
- No costs

**`SECRET_KEY`:**
- Secures Flask sessions
- Random 64-character hex string
- Keep it secret!
